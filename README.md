# Doko

A docker-based development dependency manager.

## Services

- `postgres`
- `mysql`
- `redis`

## Usage

### Show a list of services

```bash
doko list
```

### Enable a service

```bash
doko enable redis
```

### Disable a service

```bash
doko disable redis
```
